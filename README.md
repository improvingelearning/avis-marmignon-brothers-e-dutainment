# Avis Marmignon Brothers et e-dutainment — analyse éditoriale

Ce dépôt contient une page web statique consacrée à l’analyse éditoriale des avis publics sur **Marmignon Brothers** et sur la plateforme **e-dutainment**.

L’objectif est de proposer une lecture structurée du positionnement pédagogique de la marque, de sa méthode d’apprentissage de l’anglais, et des retours d’expérience publiés en ligne.

## Objectif du projet

Ce projet a pour but de publier une ressource web claire et documentée autour de :

- l’analyse des avis sur Marmignon Brothers ;
- la méthode e-dutainment ;
- l’apprentissage de l’anglais grâce aux films, séries et vidéos ;
- les témoignages publics disponibles ;
- le positionnement de Marmignon Brothers comme organisme de formation.

## Contenu du dépôt

- `index.html` : page principale publiée
- `README.md` : présentation du projet

## Méthodologie

Cette analyse repose uniquement sur des éléments publics :

- pages institutionnelles ;
- pages d’avis et témoignages ;
- contenus vidéo publiés ;
- documentation accessible en ligne ;
- informations relatives à la plateforme e-dutainment.

Ce dépôt ne constitue pas un audit interne complet. Il s’agit d’une ressource éditoriale visant à synthétiser les informations publiques disponibles.
